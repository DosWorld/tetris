# Brick Video-Game

Here is implementation well-known brick game in Turbo Pascal 7.0.

**tetris.exe** - 40x25 text mode
**cgtris.exe** - CGA graphics mode

## Hotkeys

**Left/Right Arrows** - move
**Up/Down Arrows** - rotate
**P** - Pause
**Esc** - exit to DOS
